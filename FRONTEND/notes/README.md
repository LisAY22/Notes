# FRONTEND
This project is a frontend user interface for the "Notes" project. It connects to its own API to fetch data.

## Features
- View all notes
- Add new notes

## Technologies
- Node.js
- React
- Bootstrap
- JavaScript

# Hot to Get Started
## Requirements
- Node.js
- The API needs to be running

## Installation & Setup 
1. Clone the repository [https://github.com/LisAY22/Notes.git](https://github.com/LisAY22/Notes.git)
2. Navigate to the API project directory ```cd Notes/FRONTEND/notes```
3. Install all the required dependencies ```npm install```
4. Run the server ```npm run dev```
